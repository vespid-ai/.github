# vespid-ai

vespid-ai 现在公开聚焦一个项目：[`geo-skill`](https://github.com/vespid-ai/geo-skill)。

它是一个围绕 Generative Engine Optimization（GEO）的开源工具集，用来把“让网站和仓库更容易被 AI 搜索理解、引用、发现”这件事，从泛泛建议变成可执行工作流。

## geo-skill 在做什么

- 审计站点、文档、仓库在 AI-native search 中的可读性
- 生成和改进 `llms.txt`、结构化数据、页面轮廓等 machine-readable surface
- 提供可复用的 skill pack 与 Python CLI，而不是一次性的 GEO 清单
- 帮助 ChatGPT Search 等系统更准确地抽取、引用和分发公开信息

## 核心判断

GEO 不是“骗模型”的玄学，而是：

- 信息结构清晰
- 公共页面事实准确
- 元数据完整
- 检索路径稳定
- 文档和仓库对机器可读

如果这些表面做不好，再强的产品也可能在 AI 搜索里不可见。
