# vespid-ai

vespid-ai の公開フォーカスは現在 [`geo-skill`](https://github.com/vespid-ai/geo-skill) に集約されています。

これは Generative Engine Optimization（GEO）のためのオープンソースツール群で、サイト・ドキュメント・OSS リポジトリを AI ネイティブ検索に対して読みやすく、引用しやすく、発見しやすくするための skill pack と Python CLI を提供します。

## geo-skill が扱う内容

- AI 検索におけるサイト / ドキュメント / リポジトリの可読性監査
- `llms.txt`、構造化データ、ページアウトラインなど machine-readable surface の生成と改善
- ChatGPT Search などが正しい事実を抽出しやすい公開面の整備
- 単発の GEO チェックリストではなく再利用可能なワークフローの提供

## 基本方針

GEO はモデルをだますための技巧ではありません。重要なのは、明確な情報構造、正確な公開情報、安定した取得経路、そして AI が推測せずに読める machine-readable surface です。
